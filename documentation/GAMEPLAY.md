# Gameplay Guide

Learn how to play BlockChess, understand the rules, game modes, and how to earn rewards.

## How to Play

### Getting Started

1. **Connect Your Wallet**:
   - Click "Connect" in the navigation
   - Select your Sui wallet (Sui Wallet, Suiet, etc.)
   - Approve the connection

2. **Create a Game**:
   - Click "New Game" on the home page
   - Choose game mode (Solo vs HAL or Versus)
   - Configure game settings
   - Click "Start Game"

3. **Make Moves**:
   - Click a piece to select it
   - Click a valid square to move
   - Wait for opponent's move (in versus mode)
   - Or watch HAL think (in solo mode)

4. **Win the Game**:
   - Checkmate your opponent
   - Or wait for them to resign
   - Earn rewards for achievements!

## Game Modes

### Solo Mode (vs HAL)

Play against the computer opponent named **HAL** (a reference to 2001: A Space Odyssey).

**Difficulty Levels**:
- **Easy**: Beginner-friendly, makes occasional mistakes
- **Intermediate**: Balanced challenge, solid play
- **Hard**: Advanced strategy, challenging opponent. "☠️ I'm Gonna Hit You So Hard!" mode

**Features**:
- Instant moves (no waiting for human opponent)
- Practice your skills
- Earn achievements
- Adjustable difficulty

### Versus Mode

Play against another human player.

**Features**:
- Real-time multiplayer
- Password protection (optional)
- Timer support (optional)
- Public or private games

**How It Works**:
1. Player 1 creates a game
2. Player 2 joins using game ID or link
3. Both players make moves in turn
4. Game ends when checkmate or resignation

## Game Rules

BlockChess follows standard chess rules:

### Standard Chess Rules

- **Piece Movement**: All pieces move according to standard chess rules
- **Check**: King is under attack
- **Checkmate**: King is in check with no legal moves
- **Stalemate**: Player has no legal moves but is not in check (draw)
- **Castling**: King and rook can castle under standard conditions
- **En Passant**: Pawn can capture en passant
- **Promotion**: Pawn promotes to queen, rook, bishop, or knight on 8th rank

### Special Rules

- **Timer**: Optional time limit per game (if enabled)
- **Resignation**: Players can resign at any time
- **Draw Offers**: Players can offer draws (future feature)

## Rewards System

BlockChess rewards players with NFT badges for achieving milestones. All rewards are minted on the Sui blockchain as unique NFTs.

### Available Rewards

#### First Game
- **Badge**: First Game
- **Requirement**: Play your first chess game
- **Description**: Congratulations on playing your first chess game

#### First Game Created
- **Badge**: First Game Created
- **Requirement**: Create your first game
- **Description**: Congratulations on creating your first chess game

#### First Game Won
- **Badge**: First Game Won
- **Requirement**: Win your first game
- **Description**: Congratulations on winning your first chess game

#### 10 Games Won
- **Badge**: Already 10 games won
- **Requirement**: Win 10 games
- **Description**: Congratulations on your 10th victory

#### 50 Games Won
- **Badge**: Already 50 games won
- **Requirement**: Win 50 games
- **Description**: Congratulations on your 50th victory

#### 100 Games Won
- **Badge**: Already 100 games won
- **Requirement**: Win 100 games
- **Description**: Congratulations on your 100th victory

### How Rewards Work

1. **Earn Achievement**: Complete the requirement (e.g., win a game)
2. **Automatic Detection**: System detects achievement
3. **Queue for Minting**: Reward added to mint queue
4. **NFT Minted**: Badge minted on Sui blockchain
5. **Notification**: You receive notification of new badge
6. **View in Profile**: Check your rewards page

### Reward Queue

Rewards are processed asynchronously through a queue system:

- **Non-Blocking**: Continue playing while rewards mint
- **Automatic**: No manual action required
- **Reliable**: Queue persists across page reloads
- **Real-Time Updates**: See mint status in real-time

For more details, see [QUEUE.md](./QUEUE.md).

## Game Features

### Timer

Optional timer for timed games:

- **Time Limit**: Set total time per game (in seconds)
- **Countdown**: Timer counts down during your turn
- **Time Up**: Game ends if time runs out
- **Pause**: Timer pauses when not your turn

### Password Protection

Protect your games with passwords:

- **Private Games**: Only players with password can join
- **Secure**: Password required to access game
- **Optional**: Can create public games without password

### Game History

View your complete game history:

- **All Games**: See every game you've played
- **Filter**: Filter by mode, result, date
- **Replay**: Replay any game move by move
- **Statistics**: View win/loss statistics

### Move History

During gameplay:

- **Move List**: See all moves in algebraic notation
- **Navigation**: Jump to any position in the game
- **Analysis**: Review your moves

## Tips for Success

### For Beginners

1. **Start with Easy**: Play against HAL on easy difficulty
2. **Learn Basics**: Understand piece movement and value
3. **Control Center**: Control the center of the board
4. **Develop Pieces**: Get all pieces into the game
5. **Castle Early**: Protect your king

### For Intermediate Players

1. **Study Openings**: Learn common opening principles
2. **Tactics**: Practice tactical patterns
3. **Endgames**: Study basic endgame techniques
4. **Time Management**: Use timer wisely
5. **Review Games**: Analyze your games after playing

### For Advanced Players

1. **Advanced Openings**: Master specific openings
2. **Positional Play**: Understand positional concepts
3. **Complex Endgames**: Study complex endgame positions
4. **Time Pressure**: Practice under time pressure
5. **Versus Mode**: Challenge other players

## Troubleshooting

### Game Not Starting

- Check wallet connection
- Verify you have sufficient SUI for gas
- Refresh the page
- Check browser console for errors

### Moves Not Working

- Verify it's your turn
- Check if piece is selected
- Ensure move is legal
- Refresh if game seems stuck

### Rewards Not Appearing

- Check mint queue status
- Verify transaction on Sui explorer
- Wait for blockchain confirmation
- Check rewards page

### Connection Issues

- Check internet connection
- Verify Sui network status
- Refresh the page
- Reconnect wallet
